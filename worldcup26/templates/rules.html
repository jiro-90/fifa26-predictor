Rules

Points, locks, and tie-ins

Group standings

Predict the final order for every team in the group.

Exact final position: 3 points.

Off by one position: 1 point.

Anything else: 0 points.

The group locks when the first match in that group kicks off.

Match score predictions

Predict the score for each match before kick-off.

Exact home-team goals: 1 point.

Exact away-team goals: 1 point.

Correct result type (home win, draw, away win): 1 point.

Maximum per match: 3 points.

Automatic updates

Room totals update automatically after results are synced.

On cPanel, the clean setup is a cron job that runs the sync script every 5 minutes.

Important note on group tables

If your data source does not provide official standings tie-breaks, the app falls back to points, goal difference, goals scored, then alphabetical order. You can override final group positions in the tournament data file if needed.
